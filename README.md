# Task 5 - Exploratory Data Analysis (EDA)

## Objective
Perform EDA on the Titanic dataset to identify patterns, relationships, and trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data Inspection
- Missing Value Analysis
- Histograms
- Boxplots
- Scatterplots
- Correlation Heatmap
- Pairplot

## Key Findings
- Females had higher survival rates.
- First-class passengers survived more frequently.
- Fare contained significant outliers.
- Survival was positively associated with fare and negatively associated with passenger class.
